</Note>

## Language breakdown

The collection is predominantly Python. Only two folders use JavaScript:

- **JavaScript** — `allstar/` (Discord.js), `api/` (Express.js)
- **Python** — all remaining 16 non-empty directories

Within Python, most bots use `discord.py`. The exception is `main/`, which uses `discord.py-self` — the selfbot fork that authenticates as a regular user account rather than a bot token.
