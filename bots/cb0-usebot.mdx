**cb0** and **use** are two copies of the same codebase, internally branded as **usebot**. Both directories share an identical `req.txt`, the same `main.py` entry point, and the same `cogs/` structure. The bot's defining feature is a set of 30+ task loops that automatically post content from `https://api.usebot.lol/api` into registered Discord channels every 10 seconds, covering categories ranging from female/male aesthetic photos to UK drill imagery and K-pop.

<Note>
  The source contains a hardcoded API bearer token (`464099bf-6179-4cd6-9fd1-e8e5041f52c5`) used for all `usebot.lol` API calls, as well as a bot token commented out at the bottom of `main.py`. These should be treated as compromised and rotated before any deployment.
</Note>

## Dynamic prefix

Prefixes are resolved per-message via an async lookup against SQLite. Both a user-level self-prefix and a guild-level prefix are supported; the bot's own mention is always a valid fallback:

```python
async def getprefix(bot, message):
    if not message.guild:
        return "<@1171112895250169937>"
    selfprefix = guildprefix = "<@1171112895250169937>"
    async with bot.db.cursor() as cursor:
        await cursor.execute(
            "SELECT * FROM selfprefix WHERE user_id = {}".format(message.author.id)
        )
        check = await cursor.fetchone()
        if check:
            selfprefix = check[0]
        await cursor.execute(
            "SELECT prefix, * FROM prefixes WHERE guild_id = {}".format(message.guild.id)
        )
        res = await cursor.fetchone()
        if res:
            guildprefix = res[0]
    return guildprefix, selfprefix
```

A typical loop body looks like:

```python
@tasks.loop(seconds=10)
async def female():
    header = {'Authorization': 'Bearer 464099bf-6179-4cd6-9fd1-e8e5041f52c5'}
    r = requests.post("https://api.usebot.lol/api", headers=header, json={'option': 'female'})
    urll = r.json()["url"]
    async with bot.db.cursor() as cursor:
        await cursor.execute("SELECT * FROM female")
        results = await cursor.fetchall()
        for result in results:
            channel_id = result[1]
            try:
                channel = bot.get_channel(channel_id)
                await channel.send(urll)
            except:
                pass
```

## Stats task

Every 60 seconds the `stats` loop posts a system-stats embed to a hardcoded channel ID (`1143273173111033866`):

```python
@tasks.loop(seconds=60)
async def stats():
    embed = discord.Embed(color=0x2B2D31)
    embed.add_field(name="Bot", value=(
        f"Latency: `{round(bot.latency * 1000)}ms`\n"
        f"Serving: `{len(bot.guilds)} servers`\n"
        f"Monitoring: `{len(set(bot.get_all_members()))} users`\n"
        f"Commands: `{len(set(bot.walk_commands()))}`"
    ), inline=True)
    embed.add_field(name="System:", value=(
        f"CPU Usage: `{psutil.cpu_percent()}%`\n"
        f"Memory Usage: `{psutil.virtual_memory().percent}%`"
    ), inline=True)
    await slay.send(embed=embed)
```

## Running the bot

```bash
# Install dependencies
pip install -r req.txt

# Set the token in a .env file (loaded via python-dotenv)
echo "use=<BOT_TOKEN>" > .env

# Start
python main.py
```

The token is read from the `use` environment variable: `bot.run(os.getenv("use"), reconnect=True)`.
