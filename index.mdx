Skid Archive is a GitHub repository maintained by [cop-discord](https://github.com/cop-discord/skidarchive) that collects and documents the source code of Discord bots acquired through exploitation. The archive spans multiple categories: full-featured community bots (JavaScript and Python), Discord selfbots, vanity URL snipers, mass DM tools, and auto-PFP image-posting bots. This documentation covers the architecture, commands, and internals of each bot in the collection.
